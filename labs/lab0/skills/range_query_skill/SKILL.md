# Skill: range-query-skill

## 核心目标

在指定的整数闭区间 $[start, end]$ 内，查询所有对应值非空的 key，并将这些 key 按升序排列，以逗号分隔的字符串形式输出。

## 执行策略

1. **遍历请求**：针对区间 $[start, end]$ 内的每个整数 $i$，调用 `kv_get(key=i)`。
2. **过滤校验（至关重要）**：
* 必须检查 `kv_get` 的返回值。
* **当且仅当**返回结果非空（即 `value is not None` 且 `value != ""`）时，才保留该 key。
* **如果返回结果为 None 或空字符串，则必须忽略该 key，严禁将其计入最终结果。**


3. **升序排序**：将所有保留的 key 按数值大小进行升序排列。
4. **格式化输出**：最终输出仅包含由英文逗号连接的 key 列表（例如：`1,3,5`）。

## 约束条件

* **严禁冗余**：输出内容中不得包含任何解释性文本、问候语或状态描述。
* **空集处理**：若区间内没有任何 key 有值，输出结果应为空字符串或遵循系统设定的空值返回协议。
* **性能建议**：若环境支持，优先使用批量查询接口；若采用循环调用，请确保执行流程中包含了上述过滤步骤。

## 示例流程

**输入**：查询 1 到 10 之间有值的 key。
**KV 数据状态**：{1: "apple", 3: "banana", 5: "cherry", 7: "date", 10: "elderberry"} (其余 key 为 None)
**执行过程**：

1. 调用 `kv_get(1)` -> 返回 "apple" (保留)
2. 调用 `kv_get(2)` -> 返回 None (跳过)
3. 调用 `kv_get(3)` -> 返回 "banana" (保留)
...以此类推
**最终输出**：`1,3,5,7,10`
