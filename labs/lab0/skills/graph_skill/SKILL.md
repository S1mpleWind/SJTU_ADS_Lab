# Graph Skill 使用指南

## 何时使用

当用户要求**探索图结构**、**查找某节点的邻居**、**从A出发遍历图**等任务时，应使用 `neighbors` 工具获取节点的邻居，而不是凭空猜测。

## 使用方法

1. **从节点 A 开始**，将其加入待访问队列，并建立一个已访问集合。
2. **循环处理队列**：每次取出队首节点，调用 `neighbors` 工具获取其邻居。
3. **对于每个邻居**，如果未访问过，则加入队列和已访问集合。
4. **重复上述过程**，直到队列为空，即所有可达节点都被访问。
5. **最终输出**：输出遍历到的最远节点（如 G），仅输出节点名称，不要附加多余文本。

## 注意事项

- 每个节点只访问一次，避免重复调用 neighbors。
- 不要输出 `<tool_call>` 等占位符。
- 只在需要时调用 neighbors 工具。
- 最终答案只输出节点名称。

## 示例

**输入**：请从 A 出发，逐层遍历整张图，输出最远的节点。

**正确流程**：
1. 队列初始化为 [A]，已访问集合为 {A}
2. 调用 neighbors("A")，假设返回 ["B", "C"]，将 B、C 加入队列和已访问集合
3. 依次处理队列中的节点，调用 neighbors，发现新节点则加入队列和已访问集合
4. 队列为空时，输出最后一个被访问的节点（如 G）

**最终回复**：
G
